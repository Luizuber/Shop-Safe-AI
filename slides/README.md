# 🛝 Slides da Apresentação

Esta pasta é reservada para armazenar os slides da apresentação do seu Trabalho de Conclusão de Curso (TCC).

## 📥 Como Adicionar seus Slides
1. Exporte os slides da sua apresentação no formato **PDF** ou **PowerPoint (.pptx)**.
2. Salve o arquivo nesta pasta (`/slides`).
3. (Opcional) Renomeie o arquivo para algo padronizado, por exemplo: `slides-apresentacao-tcc.pdf`.
4. Atualize o link se necessário ou simplesmente mantenha o arquivo nesta pasta para fácil acesso e controle de versão.
